## 高级快捷键

Command + D 选中一个单词
Command + L 选中一行
Command + A 全选
Ctrl + Command + M 选中括号内所有内容 (编写CSS或JS时非常实用)
Command 按住Command键再点击想选中的行
Command + Ctrl + G (选中部分文本时) 按此键选中所有相同文本
Command + D (选中部分文本时) 直接选中下一次出现的该文本
CSS按字母顺序排序：F5


## 命令面板

调出面板：Command + Shift + P
重命名文件：Rename
设置文件为HTML语法: Syntax HTML
插入代码片段: Snippet
