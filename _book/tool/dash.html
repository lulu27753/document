dash

dash

API文档浏览器(API Documentation Browser) + 代码片段管理器(Code Snippet Manager)

代码片段管理器

双下划线标明占位符，在执行扩展的时候就可以通过tab键来在各个占位符之间切换

变量符号：

  • @clipboard 自动插入当前剪贴板中的内容
  • @cursor 代码片段粘贴完毕之后，自动将光标定位到此处
  • @date 自动插入当前日期
  • @time 自动插入当前时间
